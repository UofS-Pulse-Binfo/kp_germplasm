Features
========
Tripal Germplasm Importer is developed following `Tripal Developer’s Guide » Creating Custom Data Loaders <https://tripal.readthedocs.io/en/latest/dev_guide/custom_data_loader.html>`_.
This module is developed for bulk load germplasm crosses and accessions into database.
